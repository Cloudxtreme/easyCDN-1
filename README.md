## 一、简介

1. `easyCDN` 是一款cdn环境自动部署脚本，依托于Tengine

## 二、安装和使用

1、安装方法

通过Git下载（推荐）

	yum -y install git
	git clone https://github.com/xiaosong/easyCDN.git
	cd easyCDN && ./install.sh
